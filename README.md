# Lover example
Example Love2D project made with [Lover](https://github.com/Wolfyxon/lover)

## Running
```
lover run
```

## Building
`love`
```
lover build love
```
Linux
```
lover build linux
```

Windows
```
lover build win64
```
```
lover build win32
```

## Cleaning build directory
```
lover clean
```
